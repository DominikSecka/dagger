# dagger

Slovak morphological analysis and part-of-speech tagger

# Features

  - Part-Of-Speech Tagger
  - Morphological analyser
  - Build in TCP forking server

# Requirements

  - cmake
  - C++ compiler
  - lemonstring library

# Installation

  - git clone https://github.com/hladek/dagger
  - cd dagger
  - mkdir bin
  - cd bin
  - cmake ../src
  - make
  - sudo make install

# Usage

 - dagger -h for built-in help
